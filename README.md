# Simple E-commerce Backend

![alt text](image.png)

## Setup
```
$ git clone https://github.com/fsiconha/simple-ecomm.git
$ cd simple-ecomm
$ pip install poetry
$ poetry env use python
$ poetry install
```

## Run Unit Tests
```
$ poetry run python3 run_tests.py
```

## Start App
```
$ poetry run python3 run_app.py
```
